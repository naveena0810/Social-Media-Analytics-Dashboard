\# Social Media Analytics Dashboard



\## 🚀 Project Goal

This project provides an interactive dashboard built with Streamlit and Plotly to analyze engagement, posting volume, and sentiment trends from a Twitter dataset.



\## Key Findings (Based on your analysis)

\* \*\*Sentiment:\*\* The overall tone is largely positive/neutral (95% of tweets), with only a small portion being negative.

\* \*\*Engagement:\*\* Average Likes and Retweets are stable over the monitored period, showing no massive spikes outside of expected daily variation.

\* \*\*Posting Volume:\*\* Activity is consistent across the entire time frame.



\## 🛠️ Technology Stack

\* Python

\* Pandas (for data cleaning and analysis)

\* Streamlit (for building the web dashboard)

\* Plotly (for interactive visualizations)

\* TextBlob (for sentiment analysis)



\## ⚙️ How to Run the Dashboard



1\.  \*\*Install Dependencies:\*\* Make sure you are in the project folder and run the following command to install all necessary libraries:

&nbsp;   ```bash

&nbsp;   pip install -r requirements.txt

&nbsp;   ```

2\.  \*\*Run the App:\*\* Execute the main script using Streamlit:

&nbsp;   ```bash

&nbsp;   streamlit run dashboard.py

&nbsp;   ```

3\.  The dashboard will open automatically in your default web browser.
